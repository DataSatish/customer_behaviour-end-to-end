Customer Shopping Behavior Analysis – Data Analytics Portfolio Project
A complete data analysis project simulating real-world retail analytics.
• Cleaned and preprocessed raw data using Python (Pandas): handled null values, duplicates, case normalization
• Engineered new features: age groups (young adult, adult, middle-aged, senior) and purchase frequency categories
• Exported cleaned data to SQL Server for manipulation and querying
• Built interactive Power BI dashboards to visualize revenue by gender/age, top products, discount impact, shipping preferences, subscription behavior, and customer segmentation
• Provided business recommendations to improve sales, marketing targeting, and delivery efficiency
Tools: Python (Pandas), SQL Server, Power BI
